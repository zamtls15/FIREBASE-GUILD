Gmail OAuth
+ Firestore DB

Everything wired — user login, Firestore DB for all tokens & profile data, auto token refresh, and a Gmail API client with incremental mail sync. Production-ready from day one.

✓ Firebase Auth ✓ Firestore DB ✓ Gmail API ✓ Token Refresh
01

How Everything Connects

1
User clicks "Login with Google"

Firebase opens a Google popup. Gmail scopes are requested at this moment — so the user approves Gmail access during login.

2
Google returns 3 things

user (name, email, photo) + accessToken (1hr, for Gmail API) + refreshToken (forever, to silently renew).

3
Save everything to Firestore

We upsert a user document at users/{uid} with name, email, photo, both tokens, expiry time, and sync state. Single source of truth.

4
Every Gmail API call goes through tokenManager.js

Before any request it checks if the token is expired. If yes — uses the refreshToken to silently get a new one, updates Firestore, then continues. User never gets logged out.

5
sync.js stores historyId for incremental mail sync

After first sync we store Gmail's historyId in Firestore. Every poll only fetches new messages since that ID — not the whole mailbox.

04

Firestore Database Schema

Every user who logs in gets one document in the users collection. Stores everything needed to keep them logged in and sync their mail.

📁 users / {uid} / → document
FieldTypeDescription
uidstringFirebase user ID — also the document ID
emailstringUser's Gmail address e.g. john@gmail.com
displayNamestringFull name from Google profile
photoURLstringProfile picture URL from Google
accessTokenstringCurrent OAuth access token — used for all Gmail API calls
refreshTokenstringLong-lived token to silently get new access tokens when expired
tokenExpiresAttimestampExpiry time of access token (now + 1 hour)
scopestringGmail scopes granted by the user
historyIdstringGmail history ID for incremental sync — only fetch new messages
lastSyncAttimestampWhen we last ran a mail sync for this user
syncEnabledbooleanWhether background mail sync is active
createdAttimestampFirst login / account creation time
updatedAttimestampLast time this document was updated
loginCountnumberAuto-increments on each login
💡
Why store tokens in Firestore? So your backend can also call the Gmail API on behalf of the user — e.g. background sync jobs, scheduled tasks, server-side processing. The refreshToken is the critical one. Without it, users have to re-login every hour.
05

Firestore Security Rules

Deploy these rules so users can only access their own document. No one else can read your tokens.

firestore.rules
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    // Users collection — each user owns their own doc only
    match /users/{userId} {

      // Read: only the authenticated owner
      allow read: if request.auth != null
                  && request.auth.uid == userId;

      // Write: only the owner, and they cannot change their uid
      allow write: if request.auth != null
                   && request.auth.uid == userId
                   && request.resource.data.uid == userId;
    }
  }
}
Deploy rules
$ firebase deploy --only firestore:rules
✔  Deployed Firestore rules

auth.js — Login & Logout

src/auth.js
import { auth }                                   from "./firebase.js";
import { GoogleAuthProvider, signInWithPopup,
          signOut, onAuthStateChanged }            from "firebase/auth";
import { saveUserToDB }                           from "./userDB.js";

// Build provider with Gmail scopes
const provider = new GoogleAuthProvider();
provider.addScope("https://www.googleapis.com/auth/gmail.modify");
provider.addScope("https://www.googleapis.com/auth/gmail.send");

// prompt:"consent" forces Google to return refreshToken every time
// access_type:"offline" is required to get a refreshToken at all
provider.setCustomParameters({
  prompt: "consent",
  access_type: "offline",
});

// ─── LOGIN ─────────────────────────────────────────────
export async function loginWithGoogle() {
  const result     = await signInWithPopup(auth, provider);
  const credential = GoogleAuthProvider.credentialFromResult(result);

  const tokens = {
    accessToken:    credential.accessToken,
    // refreshToken is only returned when prompt:"consent" is set
    refreshToken:   result.user.stsTokenManager?.refreshToken ?? null,
    tokenExpiresAt: new Date(Date.now() + 3600 * 1000),
  };

  // Persist user + tokens to Firestore
  await saveUserToDB(result.user, tokens);

  return { user: result.user, ...tokens };
}

// ─── LOGOUT ────────────────────────────────────────────
export async function logout() {
  await signOut(auth);
  sessionStorage.clear();
}

// ─── WATCH AUTH STATE ──────────────────────────────────
export function onAuthChange(callback) {
  return onAuthStateChanged(auth, callback);
}
💡
prompt: "consent" is critical. Without it Google only gives the refreshToken on the very first login. If you miss storing it then, users have to fully re-authorize to get a new one.

userDB.js — Save & Load User from Firestore

src/userDB.js
import { db }                              from "./firebase.js";
import { doc, setDoc, getDoc,
          serverTimestamp, increment }      from "firebase/firestore";

// ─── SAVE / UPSERT USER ────────────────────────────────
// merge:true = upsert — create if new, update if exists
// Called on every login so tokens are always fresh.
export async function saveUserToDB(firebaseUser, tokens = {}) {
  const ref = doc(db, "users", firebaseUser.uid);

  const data = {
    uid:         firebaseUser.uid,
    email:       firebaseUser.email,
    displayName: firebaseUser.displayName,
    photoURL:    firebaseUser.photoURL,

    // Only update token fields if we received new ones
    ...(tokens.accessToken    && { accessToken:    tokens.accessToken    }),
    ...(tokens.refreshToken   && { refreshToken:   tokens.refreshToken   }),
    ...(tokens.tokenExpiresAt && { tokenExpiresAt: tokens.tokenExpiresAt }),

    syncEnabled: true,
    updatedAt:   serverTimestamp(),
    loginCount:  increment(1),
  };

  await setDoc(ref, data, { merge: true });

  // createdAt only on first save
  const snap = await getDoc(ref);
  if (!snap.data()?.createdAt) {
    await setDoc(ref, { createdAt: serverTimestamp() }, { merge: true });
  }
}

// ─── GET USER ──────────────────────────────────────────
export async function getUserFromDB(uid) {
  const snap = await getDoc(doc(db, "users", uid));
  return snap.exists() ? snap.data() : null;
}

// ─── UPDATE ANY FIELDS ─────────────────────────────────
export async function updateUserFields(uid, fields) {
  await setDoc(
    doc(db, "users", uid),
    { ...fields, updatedAt: serverTimestamp() },
    { merge: true }
  );
}

// ─── SAVE REFRESHED TOKENS ─────────────────────────────
export async function saveRefreshedTokens(uid, accessToken, expiresIn = 3600) {
  await updateUserFields(uid, {
    accessToken,
    tokenExpiresAt: new Date(Date.now() + expiresIn * 1000),
  });
}

// ─── UPDATE SYNC STATE ─────────────────────────────────
export async function updateSyncState(uid, historyId) {
  await updateUserFields(uid, {
    historyId,
    lastSyncAt: serverTimestamp(),
  });
}

tokenManager.js — Auto Token Refresh

⏱
Access tokens expire in 1 hour. This module checks expiry before every Gmail call. If expired — silently refreshes using the stored refreshToken and updates Firestore. User is never logged out.
src/tokenManager.js
import { getUserFromDB, saveRefreshedTokens } from "./userDB.js";

const CLIENT_ID     = import.meta.env.VITE_GOOGLE_CLIENT_ID;
const CLIENT_SECRET = import.meta.env.VITE_GOOGLE_CLIENT_SECRET;
const TOKEN_URL     = "https://oauth2.googleapis.com/token";
const BUFFER_MS     = 5 * 60 * 1000;  // refresh 5 min before actual expiry

// ─── IS TOKEN EXPIRED ──────────────────────────────────
function isExpired(tokenExpiresAt) {
  if (!tokenExpiresAt) return true;
  const exp = tokenExpiresAt?.toDate?.() ?? new Date(tokenExpiresAt);
  return exp.getTime() - Date.now() < BUFFER_MS;
}

// ─── CALL GOOGLE'S TOKEN ENDPOINT ──────────────────────
async function refreshAccessToken(refreshToken) {
  const res = await fetch(TOKEN_URL, {
    method: "POST",
    headers: { "Content-Type": "application/x-www-form-urlencoded" },
    body: new URLSearchParams({
      client_id:     CLIENT_ID,
      client_secret: CLIENT_SECRET,
      refresh_token: refreshToken,
      grant_type:    "refresh_token",
    }),
  });
  if (!res.ok) throw new Error("Token refresh failed");
  const data = await res.json();
  return { accessToken: data.access_token, expiresIn: data.expires_in };
}

// ─── GET VALID TOKEN ───────────────────────────────────
// The main export. Call this before every Gmail API request.
export async function getValidToken(uid) {
  const user = await getUserFromDB(uid);
  if (!user) throw new Error("User not found in DB");

  // Still valid → use as-is
  if (!isExpired(user.tokenExpiresAt)) return user.accessToken;

  // Expired → refresh it
  if (!user.refreshToken) throw new Error("No refresh token — user must re-login");

  const { accessToken, expiresIn } = await refreshAccessToken(user.refreshToken);
  await saveRefreshedTokens(uid, accessToken, expiresIn);

  return accessToken;
}

// ─── WRAP ANY GMAIL CALL ───────────────────────────────
// Usage: const data = await withToken(uid, token => callGmailApi(token))
export async function withToken(uid, apiFn) {
  const token = await getValidToken(uid);
  try {
    return await apiFn(token);
  } catch (err) {
    if (err.message?.includes("401")) {
      throw new Error("AUTH_REVOKED: User must re-login and re-authorize");
    }
    throw err;
  }
}

import { withToken } from "./tokenManager.js";

const BASE = "https://gmail.googleapis.com/gmail/v1/users/me";

// ─── BASE FETCH ────────────────────────────────────────
async function gFetch(token, path, options = {}) {
  const res = await fetch(`${BASE}${path}`, {
    ...options,
    headers: {
      "Authorization": `Bearer ${token}`,
      "Content-Type":  "application/json",
      ...(options.headers ?? {}),
    },
  });
  if (!res.ok) throw new Error(`${res.status} ${await res.text()}`);
  return res.json();
}

// ─── LIST MESSAGES ─────────────────────────────────────
export function listMessages(uid, query = "is:inbox", max = 20) {
  return withToken(uid, t =>
    gFetch(t, `/messages?q=${encodeURIComponent(query)}&maxResults=${max}`)
      .then(d => d.messages ?? [])
  );
}

// ─── LIST THREADS ──────────────────────────────────────
export function listThreads(uid, query = "is:inbox", max = 20) {
  return withToken(uid, t =>
    gFetch(t, `/threads?q=${encodeURIComponent(query)}&maxResults=${max}`)
      .then(d => d.threads ?? [])
  );
}

// ─── GET PROFILE ───────────────────────────────────────
export function getProfile(uid) {
  return withToken(uid, t => gFetch(t, "/profile"));
}
// ─── GET FULL MESSAGE ──────────────────────────────────
export async function getMessage(uid, messageId) {
  const data = await withToken(uid, t =>
    gFetch(t, `/messages/${messageId}?format=full`)
  );
  const h = name => data.payload.headers.find(x => x.name === name)?.value;
  return {
    id:      data.id,
    subject: h("Subject"),
    from:    h("From"),
    to:      h("To"),
    date:    h("Date"),
    snippet: data.snippet,
    body:    decodeBody(data.payload),
    labels:  data.labelIds ?? [],
  };
}

// ─── SEND EMAIL ────────────────────────────────────────
export function sendEmail(uid, { to, subject, body, isHtml = false }) {
  const type = isHtml ? "text/html" : "text/plain";
  const raw  = btoa(
    `To: ${to}\nSubject: ${subject}\nContent-Type: ${type}; charset=utf-8\n\n${body}`
  ).replace(/\+/g,"-").replace(/\//g,"_").replace(/=+$/,"");
  return withToken(uid, t =>
    gFetch(t, "/messages/send", { method: "POST", body: JSON.stringify({ raw }) })
  );
}

// ─── REPLY TO EMAIL ────────────────────────────────────
export async function replyToEmail(uid, originalMsgId, replyBody) {
  const orig = await getMessage(uid, originalMsgId);
  const raw  = btoa([
    `To: ${orig.from}`,
    `Subject: Re: ${orig.subject}`,
    `In-Reply-To: ${originalMsgId}`,
    `References: ${originalMsgId}`,
    `Content-Type: text/plain; charset=utf-8`,
    "", replyBody,
  ].join("\n")).replace(/\+/g,"-").replace(/\//g,"_").replace(/=+$/,"");
  return withToken(uid, t =>
    gFetch(t, "/messages/send", {
      method: "POST",
      body: JSON.stringify({ raw, threadId: orig.id }),
    })
  );
}

// ─── DECODE BODY ───────────────────────────────────────
function decodeBody(payload) {
  const dec = s => atob(s.replace(/-/g,"+").replace(/_/g,"/"));
  if (payload.body?.data) return dec(payload.body.data);
  const html  = payload.parts?.find(p => p.mimeType === "text/html");
  const plain = payload.parts?.find(p => p.mimeType === "text/plain");
  return (html ?? plain) ? dec((html ?? plain).body.data) : "";
}
// ─── ARCHIVE (remove from inbox) ───────────────────────
export function archiveMessage(uid, id) {
  return withToken(uid, t =>
    gFetch(t, `/messages/${id}/modify`, {
      method: "POST",
      body: JSON.stringify({ removeLabelIds: ["INBOX"] }),
    })
  );
}

// ─── MARK AS READ ──────────────────────────────────────
export function markAsRead(uid, id) {
  return withToken(uid, t =>
    gFetch(t, `/messages/${id}/modify`, {
      method: "POST",
      body: JSON.stringify({ removeLabelIds: ["UNREAD"] }),
    })
  );
}

// ─── TRASH MESSAGE ─────────────────────────────────────
export function trashMessage(uid, id) {
  return withToken(uid, t =>
    gFetch(t, `/messages/${id}/trash`, { method: "POST" })
  );
}

// ─── LIST LABELS ───────────────────────────────────────
export function listLabels(uid) {
  return withToken(uid, t =>
    gFetch(t, "/labels").then(d => d.labels ?? [])
  );
}

// ─── SEARCH ────────────────────────────────────────────
export const searchEmails = (uid, q, max = 20) => listMessages(uid, q, max);

sync.js — Incremental Mail Sync

⚡
Gmail's history API lets you do incremental sync. After the first full sync, store a historyId. Every poll only fetches changes since that ID — not the entire mailbox. Much faster.
src/sync.js
import { withToken }                      from "./tokenManager.js";
import { updateSyncState, getUserFromDB } from "./userDB.js";

const BASE = "https://gmail.googleapis.com/gmail/v1/users/me";

// ─── INITIAL SYNC ──────────────────────────────────────
// Run once after first login.
// Gets the current historyId as the starting anchor.
export async function initialSync(uid) {
  const profile = await withToken(uid, t =>
    fetch(`${BASE}/profile`, { headers: { Authorization: `Bearer ${t}` } })
      .then(r => r.json())
  );
  await updateSyncState(uid, profile.historyId);
  return { historyId: profile.historyId, total: profile.messagesTotal };
}

// ─── INCREMENTAL SYNC ──────────────────────────────────
// Call this on every poll. Only fetches what changed.
export async function incrementalSync(uid) {
  const user = await getUserFromDB(uid);
  if (!user?.historyId) return initialSync(uid);

  try {
    const data = await withToken(uid, t =>
      fetch(`${BASE}/history?startHistoryId=${user.historyId}&historyTypes=messageAdded`, {
        headers: { Authorization: `Bearer ${t}` }
      }).then(r => r.json())
    );

    const newMessages = (data.history ?? [])
      .flatMap(h => h.messagesAdded ?? [])
      .map(m => m.message);

    if (data.historyId) await updateSyncState(uid, data.historyId);

    return { newMessages, count: newMessages.length, historyId: data.historyId };

  } catch (err) {
    // historyId too old (410) → fall back to full sync
    if (err.message?.includes("410")) return initialSync(uid);
    throw err;
  }
}

// ─── POLL SYNC ─────────────────────────────────────────
// Usage: setInterval(() => pollSync(uid, handleNewMail), 30_000)
export async function pollSync(uid, onNewMessages) {
  const result = await incrementalSync(uid);
  if (result.count > 0) await onNewMessages(result.newMessages);
  return result;
}
